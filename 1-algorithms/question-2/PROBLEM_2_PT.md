# Valid Anagram

Link: https://leetcode.com/problems/valid-anagram/description/

## Problema
Dadas duas strings `s` e `t`, retorne `true` se `t` for um anagrama de `s`, e `false` caso contrário.

Um anagrama é uma palavra ou frase formada reorganizando as letras de outra palavra ou frase, usando tipicamente todas as letras originais exatamente uma vez.

## Exemplos
Exemplo 1:
```
Entrada: s = "anagram", t = "nagaram"
Saída: true
```

Exemplo 2:
```
Entrada: s = "rat", t = "car"
Saída: false
```

## Restrições

1 <= s.length, t.length <= 5 * 10^4
`s` e `t` consistem em letras minúsculas do alfabeto.

## Acompanhamento
Você consegue criar um algoritmo com complexidade de tempo menor que O(n²)?
